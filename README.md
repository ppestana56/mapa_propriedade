# Gerador de Mapas de Propriedade (PT)

Micro-utilitário em Python para converter ficheiros de GPS (GPX, KML, GeoJSON) em relatórios cartográficos profissionais.

## Funcionalidades
- Conversão Inteligente: Transforma trilhos de GPS em áreas fechadas automaticamente.
- Métricas Oficiais: Cálculo de área e perímetro no sistema de coordenadas ETRS89 / PT-TM06 (oficial em Portugal).
- Cartografia Completa: Inclui escala gráfica, seta do Norte e mapas de fundo (satélite/ruas).
- **Exportação:** Geração de ficheiros PNG e PDF de alta qualidade.

## Tecnologias
- [Streamlit](https://streamlit.io/) - Interface Web
- [GeoPandas](https://geopandas.org/) - Processamento Geográfico
- [Contextily](https://contextily.readthedocs.io/) - Mapas de Fundo

## Como usar
1. Carregue o seu ficheiro `.gpx`, `.kml` ou `.geojson`.
2. Visualize a parcela e as métricas no ecrã.
3. Gere o relatório técnico profissional.

# Property Map Generator (UK)

Python micro-utility to convert GPS files (GPX, KML, GeoJSON) into professional cartographic reports.

## Features
- Smart Conversion: Automatically transforms GPS tracks into enclosed areas.
- Official Metrics: Area and perimeter calculation in the ETRS89 / PT-TM06 coordinate system (official in Portugal).
- Complete Cartography: Includes graphic scale, north arrow, and background maps (satellite/street).
- **Export:** Generation of high-quality PNG and PDF files.

## Technologies
- [Streamlit](https://streamlit.io/) - Web Interface
- [GeoPandas](https://geopandas.org/) - Geographic Processing
- [Contextily](https://contextily.readthedocs.io/) - Background Maps

## How to use
1. Upload your `.gpx`, `.kml`, or `.geojson` file.
2. View the parcel and metrics on the screen.
3. Generate the professional technical report.

*This project was developed as a micro-service focused on simplicity and real utility for rural landowners.*
